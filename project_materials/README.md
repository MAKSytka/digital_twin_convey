# Проектные материалы

Перед финальной сдачей сюда помещаются или отсюда явно ссылаются:

- итоговый отчёт;
- презентация защиты;
- краткая спецификация системы;
- схема общей линии;
- ссылка на видеодемонстрацию;
- ссылки на CAD и другие крупные файлы.

Рекомендуемые имена:

```text
final_report.pdf
presentation.pdf
presentation.pptx
system_diagram.pdf
```

После добавления материалов обновить корневой `README.md`.
