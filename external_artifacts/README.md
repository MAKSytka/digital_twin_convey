# Внешние бинарные материалы

Крупные файлы не должны храниться в основном Git-репозитории. Для каждого артефакта заполнить таблицу.

| Артефакт | Назначение | Версия | Среда | SHA-256 | Ссылка |
|---|---|---|---|---|---|
| `Sorter.step` | CAD-модель сортировщика | заполнить | STEP | заполнить | заполнить |
| `BoxDetector.exe` | Модуль определения коробок | v1.0 | Windows | заполнить | заполнить |
| `demo.mp4` | Полная видеодемонстрация | заполнить | MP4 | заполнить | заполнить |
| Dataset | Данные машинного зрения | заполнить | каталог/архив | заполнить | заполнить |

Контрольная сумма:

```bash
sha256sum Sorter.step
sha256sum BoxDetector.exe
sha256sum demo.mp4
```

Для `BoxDetector.exe` дополнительно указать:

- входные аргументы;
- ожидаемую структуру папок;
- формат выходного JSON;
- версию модели или датасета;
- пример запуска;
- известные ограничения.
