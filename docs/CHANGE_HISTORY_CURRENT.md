# История текущей интеграции

## Этап 1. Потоковое машинное зрение

Добавлены:

- камера Gazebo;
- `vision_stream_node`;
- выделение коробок относительно фонового кадра;
- преобразование пикселей в метры;
- стабильные ID между кадрами;
- `/singulator/boxes` типа `BoxObservationArray`;
- отладочное изображение.

## Этап 2. Замкнутый контроллер сингуляризации

Добавлены:

- подписка на `/singulator/boxes`;
- формирование 56 скоростей;
- расчёт продольных проекций коробок;
- целевые межтоварные зазоры;
- коррекция угла;
- разрешение конфликтов ячеек;
- ограничение ускорения команд.

## Этап 3. Роликовое горлышко

Добавлены:

- модель сужения `760 → 600 мм`;
- два банка роликов под углом ±80° к направлению движения;
- отдельный контроллер роликов;
- боковые сходящиеся направляющие;
- новый мир и launch;
- боковая камера наблюдателя;
- команда перемещения камеры к горлышку.

## Этап 4. Исправление высоты роликов

Исправлено:

- рабочая поверхность роликов выровнена по `Z=0.080 м`;
- добавлена переходная пластина между матрицей и горлышком;
- устранён удар о первый ролик из-за ступени по высоте.

## Этап 5. Forward-only алгоритм

Исправлено:

- удалён штатный реверс товара;
- минимальная команда стала положительной;
- введён бюджет продольного отставания;
- введена эталонная траектория со скоростью `2 м/с`;
- при исчерпании бюджета отставания скорость автоматически повышается;
- входной конвейер и ролики работают на `2 м/с`;
- добавлена проверка отсутствия отрицательных команд.

## Текущее известное ограничение

При чрезмерно плотном входном потоке невозможно одновременно гарантировать большой зазор и минимальное отставание. Приоритет текущей версии:

1. не допустить движения назад;
2. не допустить столкновения со следующей волной;
3. ограничить отставание;
4. затем максимально увеличить зазор.

## Этап 6. V3: зрение и поэтапная кластерная сингуляризация

Добавлены:

- уменьшенные морфологические ядра без постоянной дилатации;
- watershed-разделение соприкасающихся контуров;
- более чувствительный порог фонового вычитания;
- устойчивое сопоставление треков с поперечным штрафом;
- постоянные ранги внутри кластера по координате Y;
- boost-зона первых трёх рядов;
- межкластерное дозирование потока;
- усиленный finalizer последних трёх рядов;
- диапазон положительных скоростей `0.10–3.00 м/с`.


## V7 — immutable global queue

- удалён абсолютный планировщик временных слотов выхода;
- удалено повторное создание кластеров в середине матрицы;
- добавлена единая неизменяемая очередь товаров;
- добавлен прямой регулятор зазоров соседних товаров;
- добавлена логическая идентичность поверх raw-ID машинного зрения;
- добавлено сопровождение отдельных ghost tracks при объединённом контуре;
- общая ячейка теперь получает взвешенную команду всех контактирующих товаров;
- yaw-коррекция отключается в плотной паре;
- добавлены метрики инверсий, неразделённых товаров на выходе и управляемости контакта.
